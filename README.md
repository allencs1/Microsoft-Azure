## 📌 Projects

### 🧱 Project 1: Terraform VM Deployment

* Deployed an Azure Linux VM using Terraform
* Demonstrates Infrastructure as Code (IaC) fundamentals

---

### 🌐 Project 2: VNet Networking Lab

* Configured virtual networks and subnet communication
* Tested connectivity between resources

---

### 🔐 Project 3: IAM Access Control

* Implemented role-based access control (RBAC)
* Tested permissions using different users

---

### 🏗️ Project 4: Multi-Tier Architecture

* Built a multi-tier environment with frontend/backend communication
* Demonstrates layered cloud architecture

---

### 🛡️ Project 5: Defense-in-Depth (Security)

* Secured VM access using NSG rules
* Restricted SSH to a single trusted IP
* Implemented SSH key authentication
* Created custom admin user
* Verified access control with real testing
